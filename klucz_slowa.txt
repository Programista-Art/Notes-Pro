Database programmer - designs and manages data sets for specific domains, e.g., human resources, payments.
Application programmer - creates software with various applications, including text editors, graphic programs.
Tool programmer - develops system programs (e.g., operating systems) and specialized programs (e.g., controlling production processes).
Web application programmer - creates, maintains, and expands internet services.
Computer communication systems programmer - writes programs used in data transmission.
Application tester - verifies the correctness of the written program.
Project manager - manages the creation of the entire program at every stage of its development.
Application designer - based on data gathered from the client, prepares the application algorithm and systematizes data access.
Application analyst - responsible for client communication, analyzes and presents their needs to the team.
